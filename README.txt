Premium Birthday Storybook

Included
- index.html
- style.css
- script.js
- photos/ folder
- music.mp3 placeholder

Setup
1. Replace files in photos/ with your images named:
   photo1.jpg
   photo2.jpg
   photo3.jpg
   photo4.jpg
   photo5.jpg
   photo6.jpg (optional)
   photo7.jpg (optional)
   photo8.jpg (optional)

2. Replace music.mp3 with your own subtle instrumental Happy Birthday mp3.

3. Upload everything to a GitHub repo.

4. In GitHub:
   Settings > Pages > Deploy from a branch > main > /root

Notes
- Music starts after the first click.
- Final chapter includes an animated envelope and letter reveal.
- Works well on desktop and mobile.
